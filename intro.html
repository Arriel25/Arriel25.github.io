Game Introduction

Elemental Fighting Master Game Setting

Elemental Theme: Players choose an element as their character, such as Fire, Water, Air and Earth.

Keyboard Controls: Players use the keyboard to control their character's movement, attack and release skills.

Core Functional Modules

  • Character selection interface
    Player selects an element and the page dynamically loads the character model and special effects.
  • Battle Interface
    Player 1 controls (keyboard WASD):
    W: Jump
    A: Move left
    D: Move right
    S: Defense
    F: Normal Attack
    G: Release Skills
    Player 2 controls (arrow keys):
    Up: Jump
    Left: Move left
    Right: Move right
    Down: Defense
    Numeric key 1: Normal Attack
    Numeric key 2: Release Skill
  • Scoring System
    Each time you hit your opponent, your score will be +10. Consecutive hits will trigger the "Combo Bonus".
